Project 1—Merge Animation & Audio

Aim of Project

To find a method of playing back a CSS animated chord-chart along side an accompanying MP3 audio file with just one click.

The following is a typical example of a chord-chart:

These charts are very useful to musicians because they map-out exactly how and when the chords of a song (say) change.

The following audio file shows how a guitarist might use the chart to add a simple accompaniment.

When a beginner changes chord, a lot of attention is given to the fingers, which makes following a chart even more difficult. A moving cursor would be a huge help. Tap (or mouse hover) the following chart.

This CSS animated chart actually works perfectly well with the MP3 file above. Problem is, they're each activated separately.

(With a bit of practise it's possible to get both playing together: click the audio 'play' and immediately tap or hover over the animation.)

The following chart and music file are a little more sophisticated. There's a four-click intro before the chords start playing, allowing time to pick up the guitar and play along.

The other change included in this chart is a return to the first chord, A, after the fourth strum on E. It makes the progression sound more complete.
