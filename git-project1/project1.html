Project 1—Merge Animation & Audio

Aim of Project

To find a single-action method of playing back an animated chord-chart and its accompanying audio file.

The following is a typical example of a chord-chart:

These charts are very useful, especially for budding musicians—and musicians who like to improvise a bit—because they map-out exactly how and when the chords of a song (say) change over time.

The following audio file shows how a guitarist might use the above chart to add a simple accompaniment.

Beginners spent a lot of time monitoring the hand that changes chord, which makes following a chart a little more tricky. However, a simple moving cursor helps keep track of where the music is. Tap (or mouse hover) the following chart.

This animated chord-chart actually works perfectly with the music file above. The only problem is, they both have seperate triggers. It is possible to view them together, but it means clicking the audio file, and then quickly tapping (hovering) the animation.

The following chart and music file are a little more sophisticated. There's a four-click intro before the chords start playing, allowing time to pick up the guitar and play along.

The other change included in this chart is a return to the first chord, A, after the fourth strum on E. It makes it sound complete.
